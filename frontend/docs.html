RSS-Watchdog Documentation

What is RSS-Watchdog

RSSWatchdog is a keyword-based RSS monitoring tool that watches RSS feeds for content you care about.

Create a monitor by providing:

  • An RSS feed URL
  • A keyword
  • A notification method

When a monitor runs, RSSWatchdog fetches the feed, scans recent entries for your keyword, and triggers the selected action if a match is found.

Supported actions:

  • Email:- send a notification to an email address
  • Webhook:- send a JSON payload to a webhook URL
  • Log Only:- save the result without sending a notification

How it works:

  1. Create a Monitor -> Enter a feed URL, keyword, and notification method.
  2. Run the Monitor -> RSSWatchdog fetches the RSS feed, parses the latest entries, and checks for keyword matches.
  3. Trigger an Action -> If a match is found, RSSWatchdog sends an email, calls a webhook, or records the result in the logs.
  4. View Logs -> Every run is saved with a timestamp, status, and any matching entry information.

Use Cases:

Tech News Tracker

Monitor a news feed like TechCrunch and get notified when articles mention keywords such as "AI" or "Python".

Podcast Alerts

Watch a podcast RSS feed for topics, guests, or episode keywords you're interested in.

Software Release Monitoring

Track release or changelog feeds for specific versions, features, or project updates.
